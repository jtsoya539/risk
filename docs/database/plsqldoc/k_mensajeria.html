Package k_mensajeria

Agrupa operaciones relacionadas con el envío de mensajes a los usuarios
 
El envío de mensajes se puede realizar a través de:
  • Correo electrónico (E-mail)
  • Mensaje de texto (SMS)
  • Notificación push
